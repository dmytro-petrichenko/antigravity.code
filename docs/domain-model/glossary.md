# Glossary / Ubiquitous Language

## General
*   **Bounded Context**: A linguistic and architectural boundary within which a specific model is valid.
*   **Domain Event**: A record of something meaningful that happened in the domain.

## Math Domain
*   **Formula**: The user-defined algebraic rule (e.g., `z = x * y`).
*   **Grid**: The data structure representing the calculated surface.
*   **Range**: The min/max extent of the coordinate system.

## Scene Domain
*   **Scene Graph**: The hierarchy of visual nodes.
*   **Mesh**: The 3D geometry generated from the Grid.
*   **OffscreenCanvas**: A canvas element detached from the DOM, allowing rendering in a Worker.

## Interaction Domain
*   **Intent**: The abstract goal of a user action (e.g., "Rotate View", "Change Scale").
